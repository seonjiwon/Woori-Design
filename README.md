
## 프로젝트 구조
```
src/
 ├─ App.jsx
 └─ components/
    ├─ card/
    │  ├─ Card.jsx                // 전체 컨테이너 컴포넌트
    │  ├─ CardBack.jsx            // 카드 뒷면 컴포넌트
    │  ├─ CardFront.jsx           // 카드 앞면 컴포넌트
    │  ├─ CardHistoryList.jsx     // 카드 사용 내역 리스트 컴포넌트
    │  └─ CardHistoryItem.jsx     // 단일 카드 사용 내역 컴포넌트
    └─ ui/
       ├─ CardBadge.jsx
       ├─ CardInfo.jsx
       └─ FlipButton.jsx
```
